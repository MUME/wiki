The **pour** command can be used to transfer a fluid from one container
to another.

Syntax:

```
pour source destination
pour container out        (alternative to emptying)
```

Example:

`> pour fountain skin`

See also: [Drink](./Items.md#drink), [Empty](./Empty.md)
