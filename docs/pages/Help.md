**Syntax:** `help [topic]`

The **help** command in [MUME](./MUME.md) displays help files on various topics directly in-game. It is the fastest way to look up command syntax or mechanic details without leaving the game.

- `help` — displays a top-level index of help topics.
- `help <topic>` — displays the help file for the named topic (e.g., `help consider`, `help cast`).
- `help skills` — lists all available skills and their descriptions.

## This wiki vs in-game help

This wiki expands on the in-game help files with additional context, player tips, identified item lists, and guides that go beyond what the in-game system covers. Use `help` in-game for quick syntax reminders; use the wiki for strategic guidance and comprehensive references.
