**Control Weather** is a [level](./Level.md) 10 [mage](./Mage.md) [spell](./Spell.md). The main use of this spell is to create lightning storms to enable [Call Lightning](./Call_Lightning.md).

This spell will change the "course" of the current local [weather](./Weather.md). \<effect\> describes the desired result, possible effects are:

```
 clouds more          temperature lower         fog increase
 clouds less          temperature higher        fog decrease
```

This spell can be learned from:

| Guild | Sessions |
|-------|--------|
| [Forlond Mage Guild](./Forlond_Mage_Guild.md) | 12 |
| [Lórien Mage Guild](./Lórien_Mage_Guild.md) | 16 |
| [Rivendell Mage Guild](./Rivendell_Mage_Guild.md) | 3 |
