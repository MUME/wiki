The basic **direction** commands move your character between [rooms](./room.md). Each costs [movement points](./Movement_points.md) depending on terrain and encumbrance.

| Command | Abbreviation |
|---------|-------------|
| north   | **n**       |
| south   | **s**       |
| east    | **e**       |
| west    | **w**       |
| up      | **u**       |
| down    | **d**       |

If a direction is blocked by a closed door, you must open it first — or [knock](./Knock.md) if it is locked. Some directions are only available in certain [room](./room.md) types (e.g., **up**/**down** in multi-level buildings or caves).
