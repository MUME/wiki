**Pandora** is a popular graphical [MUME](./MUME.md) client known for its built-in auto-mapper ([PandoraMapper](./PandoraMapper.md)), which automatically tracks your position on the MUME map as you move. It provides a full graphical interface with customizable panels for health/mana bars, room descriptions, and combat output.

Pandora is one of the most commonly recommended clients for new MUME players because of its integrated mapper and user-friendly interface. Experienced players also use it for the quality-of-life features it provides over raw telnet connections.

## Features

- **PandoraMapper** — automatic room tracking and map display.
- Customizable interface panels and fonts.
- Built-in trigger and alias system.
- MUME-specific protocol support.
