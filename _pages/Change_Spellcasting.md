---
title: Change Spellcasting
---

With this command you can set or show the default speed for your spells.
For example: <nowiki>g \>change spellcasting

`  _    Default spellcasting mode:`
` | |   thoroughly`
` | |   carefully`
` |~|<- normally`
` | |   fast`
` | |   quickly`
` / \ `
` _/`

\>change spellcasting q Ok.

</pre>
